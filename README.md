# Taller de Operaciones en Vectores y Matrices

Este proyecto reúne la solución a un taller sobre operaciones lógicas y estructuras de datos utilizando tres lenguajes diferentes de programación. Fue desarrollado para la materia de **Estructuras de Datos** en el programa de **Técnico en Servicios de Seguridad Informática**.

## ¿Qué contiene este kit?
* **Versión Python**: Código implementado en Python utilizando la librería integrada Tkinter para el entorno visual.
* **Versión C#**: Código de la aplicación de escritorio usando Windows Forms en .NET.
* **Versión Java**: Implementación gráfica usando la librería Java Swing.
* **Propósito**: Demostrar el uso y la implementación de vectores, matrices y almacenamiento en memoria a través de diferentes lenguajes gráficos.

## Archivos del repositorio
* `python/`: Script y entorno visual de Python.
* `csharp/`: Código fuente de formularios y archivo del proyecto C# (sin carpetas temporales bin/obj).
* `java/`: Archivo de código en Java.
